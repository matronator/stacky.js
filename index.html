Stacky.js

About

Stacky.js is a simple JavaScript module for creating beautiful scrolling navigation lists with stacking headers.

Scroll this page to see it in action!

Features

  • Vanilla JS without any dependencies
  • Lightweight at around 1KB minified
  • Refreshes on window resize and DOM mutation
  • Compatible with the latest version of your favorite browser

Installation

NPM

Install with npm npm install stacky.js and import in your JS file

Browser

Download the files and include in your HTML file <script src="path/to/stacky.min.js"></script>

Check out the Getting Started section of the README for example code.
